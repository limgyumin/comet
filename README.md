# 💡 Comet
Comet은 하루 중 커밋을 깜빡하여 운동장에 빵꾸가 나는 안타까운 상황을 방지하기 위해 만든 커밋 알림 프로그램입니다. 

### 사용 기술
***

##### 프론트엔드
![맞춤형 크기 – 1](https://user-images.githubusercontent.com/52942566/95589075-35858800-0a7f-11eb-8bf2-8d49e78e1cdb.png)

ReactJs, Electron, Mobx, JavaScript를 사용하여 프론트엔드를 제작하였습니다.


![맞춤형 크기 – 2](https://user-images.githubusercontent.com/52942566/95589067-33232e00-0a7f-11eb-9bbb-9a19570cc18e.png)

NodeJs, TypeScript, Typeorm을 사용하여 백엔드를 제작하였고, 데이터베이스는 MariaDB를 사용하였습니다.
